## 0. 전체 흐름

```
[일상] → 만두가 산책 조름(하루 1회 강제) → 산책 미니게임
       → (점수 따라) 더러워짐 → 씻기 → 말리기 → 빗질 → 보상(말풍선)
       → [일상]
```

기존 간식→똥→치우기 루프와 짝이 되는 두 번째 케어 루프.

## 1. 산책 미니게임 (확정: A. 리드줄 드래그)

### 컨셉
화면 중앙에 만두 고정. 좌우 드래그로 만두 위치/방향 조정해서 다가오는 장애물을 피하고, 떨어진 간식을 줍는다.

### 인터랙션
- **만두 위치 조작**: 마우스/터치 드래그 좌우 (기존 드래그 코드 재활용)
- **장애물**: 자전거 🚲, 웅덩이 💧, 다른 강아지 🐕, 낙엽 더미 🍂
- **수집물**: 간식 🦴, 하트 💛
- **종료 조건**: 일정 시간(예: 30초) 경과 또는 hp/스트레스 게이지 소진

### 점수 → 결과 매핑
| 결과 | 조건 | 더러움 정도 |
|------|------|-------------|
| 우수 | 장애물 1회 이하 충돌 | 진흙 2~3개 |
| 보통 | 장애물 2~4회 충돌 | 진흙 4~5개 |
| 못함 | 장애물 5회+ 충돌 | 진흙 6~8개 + filter sepia 강도↑ |

→ **무조건 더러워짐**(스킵 불가). 못 할수록 더 더러움.

### 자산
- **새 이미지 0장**. `mandu_up.png` 그대로 사용, 환경은 이모지 + CSS.

## 2. 산책 트리거 규칙 (확정)

### 산책 버튼
- **상시 노출**. 사용자가 언제든 누를 수 있음.

### 만두가 조르기 (하루 1회 강제 이벤트)
- **하루에 한 번 무조건** 만두가 "산책 가자!" 말풍선으로 조른다.
- 조르는 동안에는 **다른 인터랙션 잠금**:
  - 쓰다듬기 ❌
  - 간식 주기 ❌
  - 미니게임(다른 거) ❌
  - → 오직 산책만 가능
- 조르기는 무시할 수 없는 모달성. 산책 가야 풀린다.

### 횟수 제한 (확정)
- 하루에 **여러 번 산책 가능**.
- 단, **3번째 시도부터** 만두가 화낸다:
  - 말풍선: "피곤한데 왜 자꾸 나가자고 해...", "오늘은 그만!", "다리 아파ㅠㅠ"
  - 산책 거부. 게임 진입 차단.
- 카운트는 자정 기준 리셋. 🟡 TBD: "하루"의 기준이 실제 자정인지, 앱 첫 실행 후 24시간인지.

### 데이터 저장
- 일일 카운트 + "오늘 졸랐는지" 플래그 → localStorage.
- 키 후보: `mandu.walk.count.YYYYMMDD`, `mandu.walk.lastNagDate`.

## 3. 더러워짐 표현 (자산 0장)

### 비주얼
- `mandu_up.png` 위에 CSS `filter: sepia(0.4~0.7) brightness(0.85)` (강도는 점수 따라)
- **진흙 이모지 오버레이** 3~8개를 만두 위 랜덤 위치에 spawn (똥 시스템과 코드 구조 동일하게 재활용)
- 진흙 후보: 🟤 🍂 🌿 (💩 은 똥과 겹쳐서 제외)

### 상태 머신
- `isDirty` 플래그.
- 더러운 상태에서는:
  - 쓰다듬기 → "씻고 나서!" 말풍선 (혹은 그냥 약하게 반응)
  - 간식 주기 → 가능 🟡 TBD (또는 차단?)
  - 다른 미니게임 → 차단

## 4. 씻기 → 말리기 → 빗질 시퀀스 (확정: 강제)

순서 강제. 건너뛰면 만두가 거부.

### 4.1 씻기
- 진입: 산책 직후 자동 모달 (똥 처리처럼 강제 흐름).
- 인터랙션: **만두 위에서 드래그(문지르기)**. 드래그 거리 누적.
- 시각: 거품 🫧 파티클이 위로 솟음. 진흙 이모지가 하나씩 사라짐. sepia filter 점진적으로 풀림.
- 완료 조건: 진흙 이모지 전부 제거 + sepia=0.
- 완료 후: 자동으로 다음 단계(말리기).

### 4.2 말리기
- 인터랙션 후보 (둘 중 택1 - 🟡 구현 시 결정):
  - **(a) 수건 좌우 빠르게 스와이프** N회 (15~20회)
  - **(b) 드라이기 💨 버튼 길게 누르기** (progress bar, 3~4초)
- 시각: 만두에서 물방울 💧 사방으로 튐, 살짝 wobble.
- 게이트: 말리기 안 하고 빗질 시도 → "아직 물 뚝뚝 떨어져!" 말풍선 + 거부.

### 4.3 빗질
- 인터랙션: 만두 위 **위→아래 swipe 반복** (8~10회).
- 시각: 빗 🪮 커서, 스와이프마다 ✨ 🐾 파티클.
- 완료: 큰 하트 6개 + 보상 말풍선.

### 4.4 게이트(강제 시퀀스) 메시지 예시
- 씻기 안 함 → 비누 묻은 상태로 말리기 시도: "비누 다 헹궈줘!"
- 말리기 안 함 → 젖은 상태로 빗질 시도: "물 뚝뚝 떨어져ㅠㅠ"

## 8. 구현 우선순위 (ralph 돌릴 때 참고)

1. **산책 버튼 + 미니게임 진입/종료 골격**
2. **리드줄 드래그 게임플레이** (장애물, 충돌, 점수)
3. **점수→더러움 매핑 + 더러움 비주얼** (CSS filter + 진흙 이모지)
4. **씻기 인터랙션**
5. **말리기 인터랙션**
6. **빗질 인터랙션**
7. **시퀀스 강제 게이트 + 거부 말풍선**
8. **하루 1회 조르기 이벤트 + 다른 인터랙션 잠금**
9. **3회 초과 시 화내기**
10. **풀시퀀스 완료 보상 말풍선**
11. **localStorage 영속화 (카운트, 마지막 조른 날짜)**
