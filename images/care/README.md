# 케어 시퀀스 도구 이미지 자산

씻기 → 수건닦기 → 드라이기 → 빗질 흐름에서 사용자가 잡고 끌고 다니는 **도구**들의 이미지입니다.
이미지가 없으면 자동으로 이모지 폴백(🧻 💨 🪮)으로 표시됩니다.

## 파일 목록

### `towel.png` — 수건 (수건닦기 단계)

- **용도**: 사용자가 잡고 만두 위에서 비비는 도구. "물기를 닦아내는" 느낌.
- **권장 크기**: **128x128 px 이상 정사각** (게임에서 100x100 으로 렌더되지만 레티나 대비 1.5~2x 권장).
- **포맷**: **투명 PNG**. 배경 투과 필수 (만두 위에 떠다니는 형태).
- **시점**: 위에서 본 평면 수건. 살짝 접힌 모양 또는 동그랗게 말린 모양 OK.
- **색감**: 만두/풀밭 배경(`#FFF4E0` 모달, 만두 베이지/브라운)과 잘 어울리는 톤. 너무 어두우면 안 보임.
- **음영**: 약간의 자체 그림자 정도. 게임에서 추가 drop-shadow 가 들어갑니다.

### `dryer.png` — 드라이기 (말리기 단계)

- **용도**: 사용자가 잡고 만두 위에 가져다 대는 도구. "따끈한 바람"으로 말리는 느낌.
- **권장 크기**: **128x128 px 이상 정사각**.
- **포맷**: **투명 PNG**.
- **시점**: 옆모습 또는 사선. 노즐(바람 나오는 쪽)이 화면 아래 또는 좌측을 향하도록 그리면 자연스럽습니다 (만두 위에 대고 있는 모습).
- **색감**: 분홍/오렌지/하늘색 등 분명한 색. 수건이랑 시각적으로 확연히 구분되어야 합니다.

### `comb.png` — 빗 (빗질 단계)

- **용도**: 사용자가 잡고 만두 위에서 위→아래 방향으로 끄는 도구.
- **권장 크기**: **128x128 px 이상 정사각**.
- **포맷**: **투명 PNG**.
- **시점**: 빗살이 아래쪽을 향하도록 세로로 그리는 게 자연스럽습니다 (위→아래 드래그 방향성과 맞춤).
- **색감**: 나무색/파스텔 톤. 빗살이 또렷이 보이도록.

## 인터랙션 시각적 차별화 (참고)

| 도구 | 동작 | 화면 변화 |
|------|------|-----------|
| 수건 🧻 | 만두 위에서 비비기(드래그) | 💧 물방울이 점점 줄어 0 으로 |
| 드라이기 💨 | 만두 위에 가만히 대기(hold) | 💨/〰️ 수증기 파티클이 만두에서 위로 떠오름 |
| 빗 🪮 | 위→아래 방향으로 드래그 | ✨/🐾 반짝 파티클 |

## 로딩 폴백

이미지 파일이 없으면(또는 로드 실패하면) 자동으로 이모지로 표시됩니다 — 게임 동작에는 지장 없습니다.
이미지를 채워두면 시각적으로 훨씬 명확해집니다.
